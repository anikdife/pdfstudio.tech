# PDF Editor (Phase 1 skeleton)

Vite + React + TypeScript browser-only PDF editor scaffold.

## Run

- `npm install`
- `npm run dev`

Then open the printed local URL (typically http://localhost:5173).

## Notes

- Local file only (no backend).
- Rendering uses `pdfjs-dist`.
- Editing state uses Zustand.
- Export uses a `pdf-lib` pipeline stub (no real overlay stamping yet).
